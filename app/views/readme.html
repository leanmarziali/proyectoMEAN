Creación de Admins

Teniendo en cuenta el enunciado del proyecto, se asume que los administradores están precargados en la app.

Para cargar un admin ejectuar la siguiente query en el cliente mongodb conectado a la base de datos:

db.users.insert( { 'username' : 'Mariano Tucat', 'rol' : 'ADMIN', 'local' : { 'email' : 'mt@cs.uns.edu.ar', 'password' : '$2a$04$mk05IympiPNQyIPybtfReu1s.U7/021bCmGT6e8wtipmfz/YPlMLK' } } )

Para este ejemplo se considera que la contraseña es 'admin'. Se usa bcrypt para hashear las pw planas por lo que en el ejemplo se observa el resultado de la contraseña 'admin' codificada con bcrypt.

En el método generateHash del modelo user, se especifica como se genera el hash. Para la codificación se debe hacer uso de bcrypt con 8 pasadas(Salt = 8).
